# Amazon_sales_analytics
End-to-end e-commerce data analytics project with Python, SQL, and Power BI dashboards.
---
## 🛒 Amazon India Sales Analytics (2015–2025)

📊 Project Overview

This project presents a complete end-to-end e-commerce analytics solution built on 10 years of Amazon India transactional data. It covers the full data pipeline — from raw data cleaning and preprocessing to SQL database design and interactive business intelligence dashboards — with the goal of generating actionable insights for data-driven decision-making.

---

🎯 Objectives

- Clean and preprocess large-scale, real-world messy data
- Perform in-depth exploratory data analysis (EDA)
- Design and optimize a SQL database for analytics
- Build interactive dashboards for business insights
- Enable data-driven decision-making through visualization

---

🛠 Tech Stack

- Python: Pandas, NumPy, Matplotlib, Seaborn
- SQL: Database design, queries, indexing
- Visualization: Power BI / Streamlit
- Tools: VS Code, GitHub,

---

🔄 Data Pipeline

1. Data Collection
   
   - Raw transactional datasets (2015–2025)

2. Data Cleaning & Preprocessing
   
   - Handled missing values, duplicates, and inconsistencies
   - Standardized formats (dates, prices, categories)
   - Outlier detection and correction

3. Exploratory Data Analysis (EDA)
   
   - Revenue trends and growth analysis
   - Customer segmentation (RFM)
   - Category and product performance
   - Seasonal and geographic insights

4. SQL Database Integration
   
   - Designed normalized schema (transactions, customers, products, time dimension)
   - Implemented optimized queries and indexing
   - Connected database to dashboard

5. Dashboard Development
   
   - Built interactive dashboards with 40+ visualizations
   - Enabled filtering and drill-down analysis
   - Created executive-level summary views

---

📈 Key Features

- 📊 40+ business intelligence visualizations
- 🔍 Interactive filtering and drill-down capabilities
- 🧠 Customer segmentation using RFM analysis
- 📉 Revenue, category, and product performance analysis
- 🌍 Geographic and seasonal trend insights
